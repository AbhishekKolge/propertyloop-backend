# Propertyloop Backend using Node JS

It is a backend for Propertyloop web application. It is an app where user can register as tenant or landlord and post or apply for a property.

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`FRONT_END_ORIGIN`

`CLOUD_NAME`

`CLOUD_API_KEY`

`CLOUD_API_SECRET`

`JWT_SECRET`

`MONGO_URI`

`EMAIL_FROM_NAME`

`EMAIL_FROM_ID`

`SENDGRID_HOST`

`SENDGRID_PORT`

`SENDGRID_USER`

`SENDGRID_API_KEY`

`TOKEN_EXPIRATION_TIME`

`NODE_ENV`

`TEST_TENANT_ID`

`TEST_LANDLORD_ID`

`PORT`

## Run Locally

Clone the project

```bash
  git clone https://github.com/AbhishekKolge/propertyloop-backend.git
```

Go to the project directory

```bash
  cd propertyloop-backend
```

Install dependencies with NPM

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
